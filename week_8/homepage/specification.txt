JavaScript
The website includes several interactive features powered by JavaScript:
-> Form validation and submission handling with feedback modal
-> Dynamic project card generation in the projects page
-> Fade-in animations for page elements
-> Interactive buttons with click handlers
-> Bootstrap modal functionality for the contact form

Bootstrap Features
The website utilizes multiple Bootstrap components and features:
-> Responsive navigation bar with mobile menu
-> Grid system for layout
-> Cards for content containers
-> Progress bars for skills display
-> Modal dialog for form submission feedback
-> Form styling and validation
-> Buttons and button styles
-> Responsive breakpoints for mobile optimization
